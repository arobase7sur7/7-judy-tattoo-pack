## üìÅ **Installation**

### **1. Add the resource to your server**

Place the folder:

```
7-judy-tattoo-pack
```

into your `resources` directory.

Then add this to your `server.cfg`:

```
ensure 7-judy-tattoo-pack
```

### **2. Illenium Tattoos (optional, recommended)**

Inside the pack is:

```
illenium config.lua
```

* Open the file.
* Copy the tattoo entries.
* Paste them into **each relative section** of your `illenium_tattoos/config.lua`.

Done ‚Äî the pack will appear as addon tattoos in Illenium.
